Hide default wordpress roles for non admins.


== Description ==

This plugin will hide default wordpress roles for non admins in users admin area.
It’s usefull if you have other users on your website that can create or modify users and you have your own custom rols.

This plugin will Hide Administrator, Editor, Author, Contributor, Subscriber roles on from the dropdown menu on (Users > Add new),(Users > All Users) and (Users > All Users > Edit User) pages.


== Installation ==

1. Install Hide Default Roles either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate the plugin through the 'Plugins' menu in WordPress.
